# Shirmani Research Paper

## Title:
Mind and Heart: A Dual-System Balance Model

## Author:
Shirmani (शिरोमणि रामपॉल सैनी)

---

## Overview

This repository presents a philosophical-structural model describing
the balance between:

- Mind (Cognitive System)
- Heart (Experiential System)

The project explores how imbalance between these two systems
affects civilization, environment, and human psychology.
